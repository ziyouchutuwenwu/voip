# 调试

## 说明

pjsua 偏全功能

sipp 偏压测

## 用法

### pjsua

安装

```sh
apt install -y build-essential libssl-dev libncurses-dev libsdl2-dev

cd /usr/local/src
git clone -b 2.17 https://github.com/pjsip/pjproject --depth=1
cd pjproject
./configure
make dep && make
make install

cp /usr/local/src/pjproject/pjsip-apps/bin/pjsua-x86_64-pc-linux-gnu /usr/local/bin/pjsua
```

启动

```sh
# id 要和 username 对应
pjsua --local-port=6000 \
  --id sip:xxx@10.0.2.1 \
  --registrar sip:10.0.2.1:5060 \
  --realm 10.0.2.1 --username xxx --password 123456 \
  --auto-answer 200
```
