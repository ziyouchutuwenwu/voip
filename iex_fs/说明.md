# 说明

## 用法

启动

```sh
pkill -f freeswitch
freeswitch -nc -nonat
```

创建 sip 用户

```sh
curl -X POST http://localhost:4000/api/users \
  -H "Content-Type: application/json" \
  -d '{"username":"1000","password":"123456","domain":"10.0.2.1"}'
```

sip 用户登录

id 要和 username 对应

```sh
pjsua --local-port=6000 \
  --id sip:1000@10.0.2.1 \
  --registrar sip:10.0.2.1:5060 \
  --realm 10.0.2.1 --username 1000 --password 123456 \
  --auto-answer 200 \
  --null-audio
```

查看在线 sip 用户

```sh
fs_cli -x "show registrations"
```

打电话

```sh
# 检查状态
curl http://localhost:4000/api/status

# 查看 sip 用户
curl http://localhost:4000/api/users


# 1000 呼叫 1001
curl -X POST http://localhost:4000/api/calls \
  -H "Content-Type: application/json" \
  -d '{"from":"1000","to":"1001"}'

# 查看活跃通话
curl http://localhost:4000/api/calls

# 挂断
curl -X DELETE http://localhost:4000/api/calls/<uuid>


# 手动拨号
pjsua
# 在 1000 里：
# 给 1001 打电话
按 m, 输入 sip:1001@10.0.2.1
# 挂断
按 h
```
