# 编译

## 说明

要带 erlang，所以要编译

## 步骤

### 源

```sh
export TOKEN=pat_1X8EQXH6EvgajWaBVWSJCG51

apt update && apt install -y curl
curl -sSL https://freeswitch.org/fsget | bash -s $TOKEN
```

### 依赖

mod_erlang_event 需要

```sh
apt install erlang -y
```

```sh
apt build-dep freeswitch -y
```

### build

下载

```sh
mkdir -p /usr/local/src/
cd /usr/local/src/
git clone -b v1.11 https://github.com/signalwire/freeswitch.git --depth=1
cd freeswitch
git config pull.rebase true
```

先运行 bootstrap, 生成 modules.conf

```sh
./bootstrap.sh -j$(nproc)
```

mod_erlang_event

```sh
sed -i 's|#event_handlers/mod_erlang_event|event_handlers/mod_erlang_event|g' modules.conf
cat modules.conf | rg mod_erlang_event
```

make

```sh
./configure --prefix=/usr/local
make
make install
```
